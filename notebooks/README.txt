Ce répertoire contient les différents notebooks et script :
- imagedatagenerator_gpyopt.py : le script permet d'utiliser GrdSearch pour trouver la combinaison optimale des paramètres pour l'augmentation de données
- exploratory_data_analysis.ipynb : analyse exploratoire du jeu de données
- gradcam_best_model_ENB7.ipynb : interprétabilité des résultats sur un modèle EfficientNetB7. Différents modèles ont été testés, voir le Streamlit pour le détail
